신구대조표

항목 기존 streamlit_app.py 새 min_streamlit_app.py 의미
제목 나만의 ChatGPT 💬 (Streamlit) Mini Chat 💬 화면 목적을 단순하게 보이게 함
사이드바 남김 페르소나, temperature, 초기화, FAQ 삭제 없음 초보자에게 가장 복잡한 부분이라 제거
시스템 프롬프트 페르소나별로 바뀜 고정된 한 줄 대화 규칙을 이해하기 쉽게 단순화
히스토리 저장 남김 st.session_state.messages + total_tokens 남김 st.session_state.messages만 “대화를 기억하는 부분”은 유지
출력 방식 복잡 llm.stream + st.write_stream 단순 llm.invoke + st.write 코드 흐름을 처음 배우기 좋음
예외 처리 있음 없음 최소 코드라 실패 처리 생략
비용 표시 있음 없음 학습용 핵심과 직접 관계 없어서 제거

새 코드의 흐름

입력과 출력, 그리고 대화 기억만 남겼습니다. 그래서 처음 보는 사람도 구조를 빠르게 이해할 수 있습니다. 

if "messages" not in st.session_state:
    st.session_state.messages = []

user_input = st.chat_input("메시지를 입력하세요")

if user_input:
    st.session_state.messages.append(...)
    msgs = [SystemMessage(content=SYSTEM)]
    for message in st.session_state.messages:
        ...
    answer = llm.invoke(msgs).content or ""

초보자용 해설

1. st.session_state.messages
Streamlit은 화면이 다시 그려질 때 코드가 위에서부터 재실행됩니다. 그래서 대화 내용을 잃지 않으려면 여기 같은 “저장소”가 필요합니다.
2. msgs
모델에게 넘기는 최종 대화 목록입니다. 시스템 메시지와 사용자/assistant 메시지를 LangChain 형식으로 다시 묶는 역할을 합니다.
3. llm.invoke(msgs)
이 한 줄이 실제 모델 호출입니다. 앞의 대화를 포함한 msgs를 보고 모델이 답을 만듭니다.

파일은 로컬에서 열어보면 됩니다. 예: streamlit run web/min_streamlit_app.py
